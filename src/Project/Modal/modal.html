MSC Health

A comprehensive healthcare booking platform built with React. It allows users to perform self checkups, instantly book consultations, manage appointments, and read health-related blogs—all powered by a JSON Server mock backend.

GitHub Live Demo

Tech Stack

  • Frontend: React, React Router, Bootstrap
  • Backend: JSON Server (mock REST API)
  • Deployment: GitHub Pages

⚙️ Key Features

  • Instant Consultation: Instantly book a doctor without an account — just enter your name and phone number.
  • Appointment Booking: Schedule with filters by specialty, choose available time slots, and cancel appointments up to 1 hour before the visit.
  • Self Checkup: A question yes/no form that gives guidance based on your answers — book a visit or explore relevant tips.
  • Doctor Reviews: Write reviews only after an appointment is marked as complete — ensuring authenticity.
  • Health Blog: Publicly accessible blog with articles and tips written by doctors — no login required.
  • Health Tips & Guide: Educational section that supplements self-checkup results with curated advice.
  • User Dashboard: Manage profile info, appointment history, reviews, and notifications all in one place.
  • Smart Notifications: Real-time alerts for pending appointments; system auto-updates status to "complete" 30 minutes after scheduled time.

Highlights

  • End-to-end experience: discovery → consultation → feedback.
  • Auto-status update for appointments (marked complete after 30 mins).
  • Separation between public and authenticated routes.
  • Responsive layout, modular components, and clean routing structure.

MSC Plant

A React + Redux e-commerce website for plant lovers. It provides smart filtering, search highlighting, and cart management with local state persistence, using JSON Server as a mock backend for product data.

GitHub Live Demo

Tech Stack

  • Frontend: React, Redux Toolkit, React Router, Bootstrap 5
  • Backend: JSON Server
  • Deployment: GitHub Pages

Key Features

  • Dynamic search with keyword highlighting.
  • Filter by category, price, and pet-friendly tag.
  • Modal view for detailed product info and quick add-to-cart.
  • Redux-powered cart with quantity update & auto-removal.
  • Contact form with local success feedback.

Highlights

  • Clean, nature-inspired UI.
  • Optimized for both desktop and mobile.
  • Full CRUD simulation via JSON Server.
  • Structured with reusable components and clear data flow.

MSC Restaurant

A modern React single-page website for restaurants featuring dynamic tabbed menu, gallery, contact, and booking sections. Includes smooth scrolling and AOS animations for an elegant user experience.

GitHub Live Demo

Tech Stack

  • Frontend: React, Bootstrap 5, AOS, Swiper
  • Backend: JSON Server
  • Deployment: GitHub Pages

Key Features

  • Tabbed food menu with hover animations.
  • Responsive image gallery (food/drink tabs).
  • Contact form with Google Map embed.
  • Online booking form with input validation.
  • Sticky header and smooth section scroll.

Highlights

  • Fully responsive, mobile-first layout.
  • Elegant, lightweight, and performance-optimized.
  • Reusable hooks for scroll & mobile menu logic.
  • Dynamic JSON data loading for menu items.

Personal Portfolio

A minimalist and responsive personal portfolio built with HTML, CSS, and JavaScript. It showcases projects, skills, and contact info using smooth animations, AOS effects, and modal-based project previews.

GitHub Live Demo

Tech Stack

  • Frontend: HTML, CSS, JavaScript
  • Library: AOS (Animate on Scroll)
  • Deployment: GitHub Pages

Key Features

  • Dynamic project modals with image carousel.
  • Responsive layout with smooth scrolling navigation.
  • Section-based animations and theme transitions.
  • Componentized HTML for easy content updates.

Highlights

  • No framework — built purely with HTML/CSS/JS.
  • Optimized for mobile and tablet devices.
  • Elegant pastel theme and clean codebase.
  • Modular and easily maintainable structure.
